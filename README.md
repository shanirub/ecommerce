# ecommerce
### A very basic e-commerce system, built using django.

---
**Apps:**
  1. `products` (modles `Product`, `Category`)
  2. `users` - (models `User`)
  3. `orders` (models `Order`, `OrderItem`)

---
Unit tests are written for each step and ran locally before pushing to repo.
